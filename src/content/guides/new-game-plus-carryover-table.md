## Quick answer

Remedy has confirmed that **Aberrant upgrades**, **health improvements**, **combat ability resource improvements**, **unlocked supernatural combat abilities**, **talents**, and **Artifacts** persist when starting New Game Plus. This makes the second run a continuation of combat build progression rather than a completely clean save.

| System | New Game Plus state |
| --- | --- |
| Aberrant upgrades | Carries over |
| Health improvements | Carries over |
| Combat ability resource improvements | Carries over |
| Supernatural combat abilities | Carries over |
| Talents | Carries over |
| Artifacts | Carries over |
| Traversal abilities | Does not carry over |

Traversal abilities reset because they are linked to story progression and the way Manhattan unfolds. The official explanation does not yet name every traversal unlock affected, so the safe rule is broader than assuming only dash, jump or levitation will reset.

## What New Game Plus adds

The mode introduces new talent nodes and lets players equip multiple different combat abilities earned from the same boss. This expands combinations that are impossible in the first playthrough, where not every option can be unlocked in a single run.

Artifact capacity also grows from three equipped slots to four. Enemies and encounters continue to evolve, and some bosses can show new behaviors. That means carryover strength is paired with increased challenge rather than a simple victory lap.

## Activities worth revisiting

Remedy specifically points to missed World Quests, bounties and collectibles, plus different outcomes in some side stories and conversations. The statement does not promise that every choice creates a new ending, so individual outcome claims should wait for the final game.
