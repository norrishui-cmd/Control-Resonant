## Quick answer

**Combat Abilities are Dylan's active supernatural powers.** Powerful story bosses called Resonants grant unique abilities, and some boss rewards branch into different choices. Using abilities consumes a Combat Ability resource; landing melee hits with the Aberrant restores it.

## How abilities are unlocked

Resonants are integrated into story progression rather than presented only as optional arena fights. Defeating one grants its unique power. Some offer branching options that shape different playstyles, and the first playthrough does not allow every option to be unlocked.

Official interface footage shows abilities assigned to active input slots. Exact slot counts, unlock order and full ability list remain subject to the final build.

## The resource loop

Melee and abilities support each other. Aberrant hits rebuild the ability resource; abilities can stun enemies; executions create a temporary melee-damage boost. A player who spends abilities without returning to melee will eventually lose access to that part of the toolkit.

## Confirmed examples

Remedy has described summon-oriented options including **Mold Turrets**, which attack independently and apply harmful status effects. Official UI imagery has also shown **Seekers**, volatile telekinetic entities that can attack and be thrown and detonated. Pre-release interface values should not be treated as final balance numbers.

## Ability choices in New Game Plus

New Game Plus allows multiple different Combat Abilities from the same boss to be equipped, creating combinations unavailable in the first run. Unlocked abilities carry over, while traversal abilities do not because they remain tied to story progression.
