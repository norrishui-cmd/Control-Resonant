## Quick answer

The official Steam listing now requires a **64-bit Windows 10 or 11 system, 16 GB RAM and 100 GB of storage**. The minimum GPU is a GeForce GTX 1070 or Radeon RX 5600 XT with 6 GB VRAM; the recommended GPU is a GeForce RTX 3070 or Radeon RX 6700 XT with 8 GB VRAM.

## Official requirements table

| Component | Minimum | Recommended |
| --- | --- | --- |
| OS | Windows 10/11, 64-bit | Windows 10/11, 64-bit |
| CPU | Intel Core i5-8500 or AMD equivalent | Ryzen 7 3700X or Intel equivalent |
| RAM | 16 GB | 16 GB |
| GPU | GeForce GTX 1070 / Radeon RX 5600 XT, 6 GB | GeForce RTX 3070 / Radeon RX 6700 XT, 8 GB |
| Storage | 100 GB available | 100 GB available |
| Sound card | TBD | TBD |

## What the table does not specify

Steam does not attach a target resolution, frame rate or graphics preset to either tier. “Recommended” therefore does not automatically mean 4K, maximum settings or ray tracing. The listing also says 100 GB of available space but does not explicitly label an SSD as mandatory.

## Can a similar GPU run it?

The requirements name reference cards, not a guarantee for every GPU with similar benchmark performance. VRAM, drivers, CPU balance and final patches matter. Laptop GPUs can also perform differently from desktop parts with the same family name.

## Steam Deck implications

The minimum hardware is substantially stronger than Steam Deck's raw GPU class, but that alone is not a formal compatibility verdict. Resolution scaling and launch optimization still matter, and Valve has not provided a Verified rating.
