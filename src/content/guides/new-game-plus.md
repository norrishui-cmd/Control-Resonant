## Quick answer

**Control Resonant includes New Game Plus.** Starting a new run carries over most combat progression: Aberrant upgrades, health upgrades, Combat Ability resource improvements, unlocked supernatural Combat Abilities, talents, and artifacts. Traversal abilities do not carry over because they remain connected to story progression and the way Manhattan opens up.

## Carry-over table

| Progress | Carries into New Game Plus? |
| --- | --- |
| Aberrant upgrades | Yes |
| Health improvements | Yes |
| Combat Ability resource improvements | Yes |
| Supernatural Combat Abilities | Yes |
| Talents | Yes |
| Artifacts | Yes |
| Traversal abilities | No |

This distinction matters when planning a second run. You begin with a stronger combat toolkit, but you cannot use late-game traversal powers to skip the story's intended world progression.

## What changes in New Game Plus

New talent nodes become available, and the ability system becomes more flexible. During the first playthrough, boss-related choices restrict what can be combined. In New Game Plus, Remedy says players can equip multiple different Combat Abilities from the same boss, creating combinations that are not possible on the first run.

New Game Plus also unlocks a **fourth artifact slot**. The first playthrough allows up to three equipped artifacts. That extra slot increases the number of passive effects and trade-offs that can be combined.

## Are enemies different?

The world continues to escalate. Remedy says enemies become more dangerous, encounters shift, and some bosses may gain new behaviors. New Game Plus is therefore more than replaying unchanged content with higher stats.

## Reasons to replay

A second run lets you revisit missed World Quests, bounties and collectibles, test new builds, and see different outcomes in some side stories and conversations. Not everything can be unlocked or necessarily discovered in a single run.
